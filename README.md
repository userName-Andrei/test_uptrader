Деплой на Vercel

[Ссылка на приложение](https://test-uptrader-steel.vercel.app/)

# Используемые технологии:

Frontend - React, Redux(thunk), React router, React Drag&Drop, Firebase;

# Функционал

1. добавление/удаление/изменение проекта;
2. добавление/удаление/изменение задачи;
3. добавление/удаление комментария к задаче;
4. добавление/удаление подкомментария;
5. изменение статуса задачи путем её перетаскивания;
6. поиск задачи по назаванию или номеру.
